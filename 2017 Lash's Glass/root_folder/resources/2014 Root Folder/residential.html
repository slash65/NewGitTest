Windows

Lash's Glass is a dealer for Cascade, International, Mercer, and now Coeur d' Alene with options in vinyl, aluminum, and wood for both windows and doors. We can replace old, drafty windows with new energy efficient vinyl windows -- retrofit windows are our specialty! With a retrofit install, you minimize the need for stucco and drywall repair, which replaces both labor costs and chance of leakage.

Showers

If you need to install, or replace a shower door, we carry a variety of shower enclosures from Wardrobe & Bath Specialties, Cardinal, and Contractors Wardrobe.

We install framed and frameless shower doors and panels, from 3/16" thick glass up to 1/2". Remember, all estimates are free!
